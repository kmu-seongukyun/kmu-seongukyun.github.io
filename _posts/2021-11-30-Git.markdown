user $ git init

## Git 저장소에서 파일의 상태
Local과 Remote

git <명령어> 를 통해서 git을 관리

## Git의 저장단위, Commit
Git에 변경사항 반영하기

### 1. 현재 Git 상태 확인
{%- highlight ruby -%}
user $ git status
----
On branch main
Untracked files:
    (use "git add <file>..." to include in what will be c)
{%- endhighlight -%}

### 2. example.py를 생성(혹은 수정)하고, 이를 Commit에 반영하고 싶은 경우
{%- highlight ruby -%}
user $ git add example.py
{%- endhighlight -%}

### 3. 변경사항이 반영된 new commit 생성
{%- highlight ruby -%}
user $ git commit -m "add example.py"
{%- endhighlight -%}

### 4. commit 기록 확인하기(Author, Commitor, Date, ...)
{%- highlight ruby -%}
user $ git log
{%- endhighlight -%}
